Hello, Hugo

2022-07-17

Hello world!

Decided to put this domain to use as a web/tech playground…and a little home on the internet.

For this site, I’m using Hugo as a static site generator (SSG)

  • In 2012, I read a great blog post making the case for static sites. At the time, I was a novice web developer mostly building WordPress sites.
  • That inspired me to use Jekyll, which my last homepage was built on. Seemed like a no brainer with the GitHub Pages native support.
  • 8 years later, it seems like Hugo has a larger community. Pretty cool. At the time of writing, I’m using the hello-friend theme.
